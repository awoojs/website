is a static site generator that is small, extendable and modular. It uses new JavaScript features and functional programming to approach static site generation from a more declarative angle.

Here's an example that converts all files in the current directory to 'hello weh!': 


const weh = require('@weh/weh')

const plugin = () => {
  return files => file.map(file => {
    file.contents = 'hello weh!'
    return file
  })
}

weh(async site => {
  site.use(plugin)
  return site
})

You can install weh using npm: 


npm install @weh/weh

More information can be found on the project's GitHub repository. This website is built using weh, and is also on GitHub.
